# Instructions  

1. Create a function called _createIntArray()_.  It has no parameters. Store the integers 1-5 into an array using a loop.  Return the array.

2. Create a function called _createArray()_.  It has 4 Strings as parameters.  Store the four string in the array in the order of their parameters.   Return the arrays.

4. Create a function called _getElement()_. It takes an integer and a String array as a parameter. Return the Nth element in an array where N is the integer provided. Return the element.

5. Create a function called _replaceElement()_.  It takes an integer, a double and a double array as parameters.  Replace the Nth element in the array with N is the integer provided.  Replace it with the double value provided.
